> 三个版本同时保留，便于对比和回看。

## v0 · original 最初版

**关键词**：黑白 / 克制 / 细线水纹 / 朴素

- 纯黑白系，米白底 `#f4f1eb`
- 顶部 hero：大字标题 + 三块"交互语义"原则卡（黑白 / 播放才苏醒 / 建筑决定水的性格）
- 每个 track：左 260px 建筑图 + 右文案
- 水流动 = canvas 黑色细线，按声音密度叠加；右上角小圆点表示 `FLOW IDLE / FLOW ACTIVE`
- 播放器 = 原生 `<audio controls>`
- 整体气质：**像一份提案 / 设计研究稿**

> ⚠️ 这一版的文件实体在中途迭代被覆盖（项目无 git）。
> 现在的 `original.html` 是基于会话上下文 + 4 个 track 的设计参数（speed / amplitude / density / ripple）重建的版本，结构和交互语义保留。

## v1 · collage 拼贴版（index.html）

**关键词**：拼贴杂志 / 牛皮纸 / 撕纸卡片 / 大水纹波形

- 米黄牛皮纸底 `#efe8d8` + 细斜纹 + 颗粒纹理
- 标题用米黄撕纸贴纸，带轻微旋转和锯齿撕边
- 关键词升级为 `.poem-grid`，**七种纸片变体**：
  1. 米黄主卡（撕边）
  2. 灰白剪报
  3. 斜体衬线 + 下划线（手写笔记感）
  4. 黑底白字
  5. 黑色下划线
  6. 深青蓝纸片（白字）
  7. 浅水蓝纸片
- 每片用 CSS 变量 `--rot` 设基础角度（±2–3°），叠加 `paperFloat` 6s 浮动 idle 动画；hover 抬升 1.08× + 暂停动画；播放态切 `paperPulse` 节奏更活泼
- "回声中的水 ," 大字短语 = 衬线贴纸 + Georgia 大逗号
- **核心视觉**：播放器背后 `.big-wave-canvas` 铺满整片，每个 track 用 theme 字符串做 seed，预生成 180 根独有声纹；播放时整条尖刺波形跟着声音能量呼吸 + 水滴飞溅 + 水光高亮带
- DOM 进度层：极细水平线 + 10px 圆点 playhead，浮在大水纹之上
- 整体气质：**像一本充满灵感的视觉笔记本**

## v3 · poster 海报版（poster.html）

**关键词**：方形海报 / 建筑图置底 / 每字独立拼贴 / 金色流光

- 每个 track 是独立的 **1:1 方形海报**，一列向下排，整页像唱片合集封面
- 整体深灰绿底（`#2a2f30 → #1f2425`），不再用米黄牛皮纸
- 建筑图**置底铺满**海报（`object-fit: cover`），上面盖两层：
  - 深灰 veil 半透明蒙层（上深下浅上深的三段渐变 + 径向暗角），保证文字可读
  - 极淡斜纹 grain，给数字感加一点手感
- 文字层四角布局：
  - 左上：`TRACK 0X` 米黄小贴纸 + 白色大标题（中文 + Georgia 英文副标）
  - 右上：时长 tag + `water idle/active`（播放时切米黄实底 + 光晕脉冲）
  - 中段：关键词群 `.chips`（整片 chip + 部分词用 `.char-group` 每字拆）
  - 中段下半：大字短语 `.phrase-line`（所有字都拆，配 Georgia 金色大逗号）
  - 中段底：大波形播放器
  - 左下：`FLOW IDLE/ACTIVE` 戳，播放时变金底深字
  - 右下：收束关键词（部分每字拆）
- **每字独立底纹** `.char-chip`：
  - 用于所有大字短语（如"回声中的水"、"清晨五点"）——每字独立旋转 ±1–2°、米黄/米白/浅灰三色循环、独立投影
  - 也用于每个 track 的"主词"（"折线"、"尖顶"、"伞盖"、"网格"）和部分收束关键词
  - 整片 chip 和 char-group 混排，既有"贴字"的密集感又不至于乱
- **播放器大幅升级**：
  - 200 根 spiky 水波填充到 `h*0.78` 高度，几乎占满整个中段（参考图那把"横刀"感）
  - 上三角尖刺 + 下倒影双层渲染
  - 叠加**金色发光 Bezier 流线**，从波中心向右延展出亮尾；以播放进度为 energyMask 中心，离中心越远越平，模拟"声音正在被点亮"
  - 第二层更宽、更淡的金光做辉光底（`shadowBlur` + 大 `lineWidth`）
  - 播放按钮 56px 实心白色大圆 + 黑三角 + 3px 光环
  - 进度条改成 2px 细线 + 米黄 playhead + 光环
- **不再包含** `.reason`（为什么是这段声音）段，彻底海报化，信息密度让位视觉
- 整体气质：**像一张会动的唱片封面 / 电影海报**

## 4 段 Track 共通的设计参数

| Track | 建筑 | 时长 | speed | amplitude | density | ripple | 性格关键词 |
|---|---|---|---|---|---|---|---|
| 01 pei | 贝聿铭感几何白墙 | 24s | 0.0007 | 6 | 10 | 2200 | 折线 / 秩序 / 留白 |
| 02 mondrian | 蒙德里安风色块网格 | 22s | 0.00135 | 10 | 13 | 1400 | 网格 / 切分 / 脉冲 |
| 03 aranya | 阿那亚礼堂清晨 | 28s | 0.00055 | 14 | 9 | 2600 | 尖顶 / 海边 / 孤独 |
| 04 mad | MAD 有机曲面鸟瞰 | 32s | 0.00045 | 16 | 8 | 3000 | 伞盖 / 呼吸 / 鸟瞰 |

参数语义：
- `speed`：水波时间步长，控制流动快慢
- `amplitude`：基础波幅
- `density`：水平流线根数
- `ripple`：涟漪生成间隔（毫秒），越小触发越频繁
